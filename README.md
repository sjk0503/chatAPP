1. 기존 생성형 인공지능과 달리, 한 가지 기능에 특화된 인공지능을 개발하여 사용자에게 정확한 정보 전달하도록 함
2. gpt api를 활용하여 Chat app을 개발

우리는 영화에 특화된 인공지능을 개발하기로 함
